Installation

Get started with Omarell on your Ubuntu system

>
Note: A new modern and minimal version is under development, which is based on Ubuntu Server and uses a different installation method.
!
Warning: The following instructions are for the legacy version of Omarell, that is based on Ubuntu Desktop. A new modern version is under development, which is based on Ubuntu Server and uses a different installation method.

Prerequisites

  • A fresh install of Ubuntu Desktop, preferably 24.04 but works with 25.04 too.
  • GNOME as Desktop Environment, although it is more recommended than necessary since it can also be installed on Ubuntu Server.
  • A stable connection, to avoid errors when downloading apps and packages

Quick Installation

To install Omarell, simply run the following command in your terminal:

wget -qO- https://omakasui.org/omarell-legacy | bash

Experimental Version

!
Warning: The development version may contain unstable features and bugs.

For contributors and brave users who want to experiment with the latest features:

wget -qO- https://omakasui.org/omarell-dev | bash
>
Note: This version is designed for a fresh new Ubuntu Server (24.04) installation, without a graphical interface and with minimal pre-installed packages.

Troubleshooting

Installation Failed

If installation fails, retry with:

source ~/.local/share/omarell/install.sh

Permission Issues

Ensure you have sudo privileges:

sudo -v
