Prerequisites

  • A fresh install of Ubuntu Server, only 24.04 LTS version.
  • A stable connection, to avoid errors when downloading apps and packages

You can follow the Ubuntu Server installation guide as suggested above. Use disk encryption only if you deem it necessary. Do not install any other packages or SSH keys; they are not needed. Once the installation is complete and you are logged in, you should leave the system as is.

Installation

To install Omarell, simply run the following command in your terminal:

wget -qO- https://omakasui.org/omarell | bash

For contributors and brave users who want to experiment with the latest features:

wget -qO- https://omakasui.org/omarell-dev | bash
!
Warning: The development version may contain unstable features and bugs.

Installation (Legacy)

A legacy version of Omarell, based on Ubuntu Desktop, is also available. To do this, simply install any version of Ubuntu Desktop and, once logged in, run the following command in the terminal:

wget -qO- https://omakasui.org/omarell-legacy | bash
!
Warning: This version is no longer considered essential to the project and may not receive frequent updates.

Troubleshooting

Installation Failed

If installation fails, retry with:

source ~/.local/share/omarell/install.sh

Permission Issues

Ensure you have sudo privileges:

sudo -v
