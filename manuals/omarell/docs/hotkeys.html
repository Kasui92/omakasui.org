Hotkeys

Hotkeys are based on Omakub and can be found here. However, due to the new features, additional hotkeys have been added to make them easier to use.

Categories

  • Window Management
  • Launching Apps

Window Management

Hotkey Action
Super + Space App launcher

Launching Apps

Hotkey Action
Super + Alt + Space Omakub menu
Super + F File Manager

Appearance

Hotkey Action
Super + Ctrl + Space Next Background
Super + Shift + Ctrl + Space Theme Launcher
>
Note: Background images live in ~/.config/omaforge/current/backgrounds. You can put more there to cycle through.
