Shell Functions

Omarell comes with a set of shell functions to simplify common tasks and encapsulate convoluted parameter calls.

Available Functions

compress [file.tar.gz] / decompress [file.tar.gz]

Create or expand a tar.gz file.

compress my-archive.tar.gz folder/
decompress my-archive.tar.gz

iso2sd [image.iso] [/path/to/sdcard]

Create a bootable drive on an SD card using the referenced iso file.

iso2sd ubuntu-24.04.iso /dev/sdb

webm2mp4 [movie.webm]

Turn the .webm movie recordings produced by Gnome's screen recording tool into a mp4 file that's easier to share online using ffmpeg.

webm2mp4 screencast.webm

web2app [Name] [URL] [URL to PNG icon]

Create a .desktop file for a web app that can then be started with wofi easily.

web2app 'GitHub' 'https://github.com' 'https://github.com/favicon.ico'

web2app-remove [Name]

Remove the .desktop file you've previously created for a web app.

web2app-remove 'GitHub'

app2folder [app.desktop] [folder name]

Move an app to a named folder.

app2folder firefox.desktop 'Browsers'

app2folder-remove [app.desktop] [folder name]

Remove an app from a named folder.

app2folder-remove firefox.desktop 'Browsers'

refresh-xcompose

Restarts fcitx5 to pickup changes made to ~/.XCompose to add new emojis or other auto completions.

refresh-xcompose

Usage Notes

>

Tip: All shell functions are automatically available in your terminal after installing Omarell. You can use them immediately without any additional setup.
