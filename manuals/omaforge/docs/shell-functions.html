Shell Functions

Omaforge comes with a set of bash functions that simplify common tasks and solve complex jobs.

Basic Functions

compress [file.tar.gz] / decompress [file.tar.gz]

Create or expand a tar.gz file.

compress my-archive.tar.gz folder/
decompress my-archive.tar.gz

iso2sd [image.iso] [/path/to/sdcard]

Create a bootable drive on an SD card using the referenced iso file.

iso2sd ubuntu-24.04.iso /dev/sdb

webm2mp4 [movie.webm]

Turn the .webm movie recordings produced by Gnome's screen recording tool into a mp4 file that's easier to share online using ffmpeg.

webm2mp4 screencast.webm

web2app [Name] [URL] [URL to PNG icon]

Create a .desktop file for a web app that can then be started with wofi easily.

web2app 'GitHub' 'https://github.com' 'https://github.com/favicon.ico'

web2app-remove [Name]

Remove the .desktop file you've previously created for a web app.

web2app-remove 'GitHub'

app2folder [app.desktop] [folder name]

Move an app to a named folder.

app2folder firefox.desktop 'Browsers'

app2folder-remove [app.desktop] [folder name]

Remove an app from a named folder.

app2folder-remove firefox.desktop 'Browsers'

ssh-key-put [user@host] [optional:path_to_key (default: ~/.ssh/id_ed25519.pub)]

Uploads the specified SSH public key to the remote host for the specified user.

Omaforge

omaforge

Shows all Omaforge features omaforge

omaforge version

Shows the current version of Omaforge and checks for new updates omaforge

omaforge update

Updates Omaforge and tracks migrations

omaforge update --system

Updates Ubuntu and cleans up orphaned packages

omaforge update --firmware

Updates the firmware

omaforge theme [theme folder name]

Updates the theme using the selected omaforge theme. If no theme folder name is provided, it will interactively select a new theme.

omaforge theme tokyo-night

omaforge apps install [app]

Installs the selected application. If no application is specified, it will interactively select one.

omaforge apps install discord

omaforge apps uninstall [app]

Uninstalls the selected application. If no application is specified, it will interactively select one.

omaforge apps uninstall discord

omaforge refresh [applications|config|gdm]

Refreshes the specified component.

  • applications: Refreshes the application list.
  • config: Refreshes the configuration files.
  • gdm: Refreshes the GDM backgrounds.
  • plymouth: Refreshes the Plymouth splash screen.

omaforge refresh applications

omaforge restart [first-boot|xcompose]

Restarts the specified service.

  • xcompose: Restarts the X11 compose file.

omaforge restart first-boot

Usage Notes

>

Tip: All shell functions are automatically available in your terminal after installing Omaforge. You can use them immediately without any additional setup.
