Installation

Get started with Omaforge on your Ubuntu system

>
Warning: Although a stable version has been released, Omaforge is still an experimental project with an unclear future. I recommend installing it to try it out, not using it on a daily basis.

Categories

  • Prerequisites
  • Installation

Prerequisites

  • A fresh install of Ubuntu Server, only 24.04 LTS version.
  • A stable connection, to avoid errors when downloading apps and packages

You can follow the Ubuntu Server installation guide as suggested above. Use disk encryption only if you deem it necessary. Do not install any other packages or SSH keys; they are not needed. Once the installation is complete and you are logged in, you should leave the system as is.

Installation

To install Omaforge, simply run the following command in your terminal:

curl -fsSL https://omakasui.org/omaforge | bash

Troubleshooting

Installation Failed

If installation fails, retry with:

source ~/.local/share/omaforge/install.sh

Permission Issues

Ensure you have sudo privileges:

sudo -v

Overview on First Boot

When you log in for the first time, even though it's disabled, you'll likely find yourself in the GNOME overview screen. In this case, just click on the workspace (any one!) and you'll finally be able to start playing.
